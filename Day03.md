## Expo CLI

- 서드 파티 서비스(무료 서비스)
- 가입 후 비용을 내면 추가 유료서비스 이용 가능하나, 없어도 된다.
- 중간에 중지하고 React Native CLI로 전환 가능하다.

## React Native CLI

- 추가적인 설정, 구성 해야함
- 편리한 기능 적다.
- 특정 네이티브 기기 기능을 활용할 때 작업이 Expo보다 더 번거롭다.
- Java, Objective-C, swift, Kotlin 등 네이티브 소스 코드와 통합하기 쉽다.
